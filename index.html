Modern Headshot

This app has an online reservation feature using React-Redux, which mimics Outlook calendar experience with graphical user interface.

React's Virtual DOM enables seamless page updates upon users real-time activities (i.e., timezone update, content update on mouse-hover, real-time input validation/normalization)

On "My Account" menu, the logged users can view or cancel their reservations or can access the outcome from the previous events with an individual routed page.

Used Technologies:
React-Redux, Redux-thunk, Redux-form, React-router-dom, Node.js, Express, Mongo DB, JWT, Passport, Rest API, Enzyme, Mocha, Chai, Travis CI

Applicant-Tracker

This app is a MVP version of HR tool, which allows HR or recruiters to track their applicants status.
The latest React release, React Hooks are used for state management, and React context API is used for shared data management among components.

All persistent data are stored in MongoDB via REST API, and users can view or edit data based upon their permission on JWT (Role-based access: Admin, Recruiter, and Normal user)

Used Technologies:
React-Hooks, React-Context-API, React-router-dom, Node.js, Express, Mongo DB, JWT, Passport, Rest API, Mocha, Chai, Travis CI

Image-Lookup

This app allows mobile phone or tablet users to analyze their local images by simply uploading them to the browser.

The user can access various file information such as file name, size, camera setting, date & time, editing software, Google map location, and weather status of where the photo was taken.

Used Technologies:
jQuery, HTML5, CSS3, JavaScript, Exif Library, Google Map API, Weather API
